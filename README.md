# Git Tutorial

## Setup

Set the name and email that will be attached to your commits and tags.
```
$> git config --global user.name  "Cagin Agirdemir"
$> git config --global user.email "caginagirdemir@gmail.com"
```
## Start a Project

Create a local repo
```
$> git init
```
or
Download a remote repo
```
$> git clone <url>
```
